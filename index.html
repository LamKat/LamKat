Welcome to the BOTL Project

About

BOTL is a reserch project to test the hyposisis that using a technological solution to reduce the cost of getting informed about urban planning will result in better engadgment in planning consultation.

This webpage is designed to make it easier to find out about what planning application have been submitted in your local area, and to facilitate better engagement.
